CSS can be added to HTML. documents in 3 ways:

  • Inline by using the style attribute inside HTML elements!
  • Internal by using a style element in the head section
  • External by using link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS clan vs ID, the difice is that CSS class applies a style in multiple elements. ID, on the other hand, upplies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specilie element or range of elements on a web page. Once an element has been targeted, a style ve set of styles can be applied to the element

There is a wide range of selectors available. Two of the must commonly used are class and ID. Both are used to target elements to which a style should be applied.
